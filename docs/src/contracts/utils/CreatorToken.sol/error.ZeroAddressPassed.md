# ZeroAddressPassed
[Git Source](https://gitlab.com/nomadhub/smart-contracts/blob/9a5d5791960776da326b790b7c18e7af6b05a3aa/contracts/utils/CreatorToken.sol)

Thrown when attempting to set a zero address as the transfer validator.

This error prevents setting an invalid address.


```solidity
error ZeroAddressPassed();
```
